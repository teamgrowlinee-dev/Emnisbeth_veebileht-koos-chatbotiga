# Emsibeth veebileht koos chatbotiga

Staatiline Emsibethi veebilehe peegel Netlify jaoks.

## Deploy

- Netlify publish directory: `public`
- Build command: ei ole vaja
- Chatbot laeb Renderist: `https://emnisbeth-chatbot.onrender.com`

## Märkused

- Veebilehe staatilised HTML failid asuvad kaustas `public/`
- Visuaalsed assetid võivad tulla jätkuvalt `emsibeth.ee` domeenilt
- Chatbot on sisse ehitatud kõikidesse peamistesse lehtedesse loaderi kaudu
